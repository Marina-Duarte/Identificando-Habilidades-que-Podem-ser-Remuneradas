# Identificando-Habilidades-que-Podem-ser-Remuneradas
Esse trabalho fez parte da conclusão do curso Identificando Habilidades que Podem ser Remuneradas, ofertado pela DIO em parceria com a Caixa Econômica Federal. Esta atividade permitiu realizar um mapeamento das competências adquiridas ao longo da trajetória profissional e acadêmica, convertendo habilidades em produtos e serviços tangíveis.
